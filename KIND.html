Jenis-jenis Tanaman Hias

Tanaman Hias Indoor


Tanaman hias indoor yang populer dan mudah dirawat meliputi:
  1. Dracaena bergaris ( Dracaena fragrans )
  2. Pohon naga ( Dracaena reflexa var. angustifolia )
  3. Papirus payung ( Cyperus alternifolius )
  4. Bromelia ( Bromeliaceae )
  5. Dieffenbachia ( Dieffenbachia ssp )
  6. Tanaman doa ( Maranta leuconeura )
  7. Cyclamen ( Cyclamen )

Tanaman Hias Outdoor


Tanaman hias outdoor untuk menambah warna dan keindahan taman Anda. Tidak hanya cantik, tetapi juga mudah dirawat. Beberapa contoh populer meliputi:
  1. Pakis ( Filicopsida )
  2. Anyelir ( Dianthus cayophyllus )
  3. Mawar ( Rosa spp .)
  4. Begoniaceae ( Begoniaceae )
  5. Maple Jepang ( Acer palmatum )
  6. Arum-lily ( Zantedeschia aethiopica )
  7. Taman petunia ( Petunia hybrida )
  8. Coleus ( Coleus blumei )

Tanaman hias Matahari


Meskipun kebanyakan tanaman senang mendapatkan sinar matahari sebanyak mungkin,
tidak semua tanaman dapat bertahan dengan paparan sinar matahari langsung yang berlebihan, terutama di daerah dengan iklim panas atau ketika sinar matahari sangat kuat. Oleh karena itu, kami menyajikan beberapa tanaman hias populer yang terkenal dapat mentolerir suhu tinggi dan paparan
sinar matahari langsung:
  1. Bambu (Bambusoideae)
  2. Echeveria (Echeveria)
  3. Bunga Transvaal (Gerbera)
  4. Aster (Aster)
  5. Dahlia (Dahlia)
  6. Bunga Raya Biasa (Hibiscus syriacus)
  7. Violet manis (Viola odorata)
